About The Project

TRAW is a typescript reddit API wrapper, a successor to Snoowrap by Not An aardvark. The main drive to start the project was to address the lack of maintenance on snoowrap, leading to out-of-date functions, and secondly, to address potential security concerns arrising from the modules upon which it relied. The opportunity was also taken to incorporate type safety by moving from javascript to typescript, leading hopefully to greater reliability presently and in the future, as well as improving the end-developer's experience by making some function calls more intuitive.
